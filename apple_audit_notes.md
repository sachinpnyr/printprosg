# Apple-Standard Audit Notes — PrintProSG

## NAVBAR
- Logo: Print Pro with ink strokes — good, recognisable. Font weight looks slightly thin at small size. NOT Apple standard — Apple logos are always perfectly balanced, no visual weight issues.
- Search bar: rounded pill shape, grey background, magnifier icon — CLOSE to Apple standard but border-radius could be slightly tighter, placeholder text colour is too light (#999 vs Apple's #8E8E93)
- Nav right: phone number + person icon + cart icon + "Get a Quote" red pill button
  - ISSUE: Cart badge shows red "0" always — Apple never shows empty badges
  - ISSUE: Phone number in nav feels cluttered — Apple would never put a phone number in the nav bar
  - ISSUE: Person icon is generic/thin — not Apple-quality iconography
  - ISSUE: Cart icon looks like a generic Bootstrap icon — not premium
  - "Get a Quote" button: red pill — colour is correct brand red but the pill shape is slightly inconsistent with the rest of the UI (other buttons are rounded-rect not full pill)
- Category tabs: text is too small (12px), underline indicator is thin red line — Apple uses bolder tab indicators or segmented controls. Tab text weight is too light.
- NAVBAR APPLE SCORE: 5/10

## HERO
- Headline "Premium Printing, Delivered Fast." — font is Impact/heavy black. This is NOT Apple standard. Apple uses SF Pro Display or similar — clean, geometric, not condensed/impact. The headline font looks like a newspaper ad, not a premium brand.
- "Delivered Fast." in red — the colour split is effective but the font choice undermines it
- Sub-text: small, grey — readable but generic
- CTAs: "Get a Free Quote" (red filled) + "Explore Products" (outlined) — the outlined button has a very thin border, hard to see. Apple uses 2px borders minimum.
- Trust row: lightning bolt + pencil + star icons with text — icons are very small (16px), look like generic emoji replacements. Apple uses SF Symbols-quality icons.
- Product cards: white cards with image + name + price — clean but the card shadow is very subtle, almost invisible. Apple uses more defined elevation.
- HERO APPLE SCORE: 5.5/10

## REVIEWS
- Section header: "WHAT OUR CLIENTS SAY" eyebrow + h2 — eyebrow text is all-caps red, looks dated. Apple never uses all-caps red eyebrow labels.
- Review cards: avatar circle + name + stars + text + Google badge — stars are red (should be gold/yellow for reviews — red stars look like warnings). Review text truncates with no ellipsis. Time stamp ("2 weeks ago") is too prominent.
- REVIEWS APPLE SCORE: 5/10

## ABOUT
- Stats: "34+ Years" "500+ Happy Clients" "50+ Products" — numbers are bold and large, good. But the "+" sign styling is inconsistent with the number weight.
- Feature cards: 4 cards with red icons (bolt, shield, layers, person) — icons are generic. The red icon on white card background is harsh. Apple uses softer icon treatments.
- CTA "Get a Free Quote" button — left-aligned under the text, inconsistent with section layout
- ABOUT APPLE SCORE: 5.5/10

## HOW IT WORKS
- 4 steps with red circular numbered icons — the red circles are too heavy/saturated. Apple uses lighter, more refined step indicators.
- Step descriptions are generic placeholder-quality text
- No visible connector line between steps
- HIW APPLE SCORE: 6/10

## SERVICES
- Service cards: image + title + price + "Get Quote" button — the card design is clean but the "Get Quote" button on every card is repetitive and clutters the grid
- Service images are good quality (real product photos)
- Section is 2554px tall on desktop — completely un-Apple. Apple pages are concise and never scroll endlessly.
- SERVICES APPLE SCORE: 4/10

## PORTFOLIO
- Clean grid with real photos — this is the strongest section
- Filter tabs are functional and well-styled
- Images are high quality
- PORTFOLIO APPLE SCORE: 8/10

## LOCATIONS
- 3 cards with flag + country + address + buttons — clean layout
- WhatsApp button is bright red — inconsistent with WhatsApp's brand green. Apple would never use wrong brand colours.
- "Fanan Advertising LLC" partner name confuses brand identity
- LOCATIONS APPLE SCORE: 6.5/10

## FAQ
- Clean accordion on grey background — functional
- Too much spacing between items (80px per row when closed)
- Only 5 questions — sparse
- FAQ APPLE SCORE: 6.5/10

## CONTACT
- Left info column + right form — good two-column layout
- Form fields: clean, well-labelled
- "Send Enquiry" red full-width button — strong CTA
- Contact info icons are small red circles — slightly heavy
- CONTACT APPLE SCORE: 7/10

## FOOTER
- Dark background with white logo — good
- Logo renders correctly with white text
- 4-column link grid — functional but text weight is too light (400) for dark background
- "Designed with ❤ for quality printing" — template leftover, not premium
- Payment badges (PayNow, Bank Transfer) look like generic pill badges, not premium
- FOOTER APPLE SCORE: 6/10

## MOBILE
- Hero: clean, headline bold, two CTAs stacked — good
- Service cards: 2-column grid, images good — acceptable
- Footer logo: too large (320px wide on 390px screen)
- Nav: hamburger menu — standard
- MOBILE APPLE SCORE: 6.5/10

## SUMMARY OF NON-APPLE ELEMENTS
1. Hero headline font — Impact/heavy black is not premium. Needs SF Pro-style geometric font.
2. All-caps red eyebrow labels — dated pattern, not Apple.
3. Red stars in reviews — should be gold/yellow.
4. Cart badge showing "0" — never acceptable in premium UI.
5. Phone number in navbar — clutters the nav.
6. Services section 2554px tall — completely un-Apple.
7. Generic icon set — not SF Symbols quality.
8. "Designed with ❤" footer line — template leftover.
9. WhatsApp button in wrong colour (red instead of green).
10. Outlined CTA button border too thin.
11. Review text truncation with no ellipsis.
12. Section header eyebrow text style inconsistent.
13. Footer logo too large on mobile.
14. About stats "+" sign styling inconsistent.
15. Cookie banner too large and intrusive.
